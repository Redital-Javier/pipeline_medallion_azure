# Documento funcional

Proyecto: Pipeline de Datos End-to-End en Azure utilizando Arquitectura Medallion

Autor: Javier Redital

## Descripción

Proyecto de ingeniería de datos desarrollado utilizando Azure Data Factory y Azure Databricks para implementar un pipeline end-to-end basado en arquitectura Medallion.

El flujo implementado contempla:

- generación de datasets
- procesamiento Bronze
- limpieza y validaciones Silver
- generación de tablas analíticas Gold
- automatización mediante Trigger

El documento funcional completo se encuentra disponible en el informe adjunto del proyecto.
